# Frontend Color Palette & UX Guidance

Recommended friendly, modern, and light color palette for ReassureAI (medical-friendly):

- Primary (Blue): #2563EB — Clear, trustworthy primary action color (buttons, links).
- Primary Light: #60A5FA — Hover / subtle backgrounds.
- Accent (Teal): #34D399 — Positive/confirm actions and badges.
- Warm (Friendly Yellow): #F59E0B — Highlights, subtle callouts, and attention (use sparingly).
- Calm (Lavender): #E9D5FF — Soft backgrounds for cards or banners.
- Coral: #FB7185 — Use for gentle warnings or error states.
- Success: #10B981 — confirmations and success states.
- Neutral text: #374151 (dark gray) and #6B7280 (muted gray) for secondary text.

UX notes:

- Use a lot of white / soft backgrounds with generous spacing.
- Buttons: primary filled with `primary` and subtle shadow; secondary as soft gray outlines.
- Cards: white background, rounded corners (1rem), subtle shadow, and `calm` tint for highlighted cards.
- Typography: large, friendly headings; readable body type (16px+).
- Accessibility: ensure 4.5:1 contrast for primary actions; use `primary-dark` for text on primary buttons.
- Chat UX: left/right message bubbles, subtle timestamp, markdown rendering, action bar with TTS/copy/regenerate.

Color rationale: blues convey trust and calm; teal adds wellbeing/health vibes; warm yellow gives friendly optimism; lavender softens the palette to feel less clinical and more human-centered.
